# SPAR-Kit Keyboard Shortcuts Reference

A quick reference guide for keyboard shortcuts in SPAR-Kit TUI Mission Control.

---

## Global Shortcuts

| Key | Action | Context |
|-----|--------|---------|
| `ESC` | Go back / Cancel | Any screen |
| `Q` | Quit / Back | Any screen |
| `?` | Show help | Any screen |
| `Tab` | Switch mode/panel | Multi-panel views |

---

---

## Tips

1. **Quick Navigation**: Use number keys (`1-9`) to jump directly to items in lists
2. **Context Help**: Press `?` on any screen to see available shortcuts
3. **Escape Hatch**: `Ctrl+C` will always exit the TUI safely
4. **Filter Presets**: In session list, press `F` followed by:
   - `A` - All sessions
   - `R` - Running
   - `C` - Completed
   - `P` - Paused
   - `F` - Failed
