#### Pattern Details

**Sequential (Ubuntu/Circle)**

**Binary (Oxford Debate)**
```
Round 1: Opening statements (FOR then AGAINST)

```

**Dialectic (Lincoln-Douglas)**
```
Thesis presented â†’ Antithesis challenges â†’ Synthesis emerges

```

**Inquisition (Scholastic)**
```

```

**Emergence (Nemawashi)**
```
Initial positions gathered
Find common ground first
Iterate toward consensus
No winners/losers, only shared understanding
```
