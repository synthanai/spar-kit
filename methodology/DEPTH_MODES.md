# Reasoning Depth Modes

*Six calibrated modes for matching debate intensity to decision stakes.*

> **Version**: 1.0 (Protocol v8.0)

---

## Overview

Not every decision needs a full 8-agent, 7-round debate. Depth modes let you calibrate the reasoning intensity to match the stakes. Each mode adjusts four parameters: number of rounds, number of personas, ABSTRACT depth, and intended use case.

---

## The Six Modes

| Mode | Rounds | Personas | ABSTRACT Depth | Use Case |
|------|--------|----------|----------------|----------|
| 👀 **Show Me** | 1 | 4 | Skip | Demo, onboarding, first-time users |
| ⚡ **Quick** | 3 | 4 | Type 1 only | Rapid sanity check, tight deadlines |
| ⚖️ **Basic** | 3 | 4 + PROBE | Types 1-2 | Moderate-stakes, standard decisions |
| 💣 **Stress Test** | 3 | 4 + PROBE | Types 1-3 | High-stakes, significant investment |
| 🧠 **Deep** | 5 | 8 + PROBE | Types 1-3 | Strategic pivots, M&A, restructuring |
| 🤯 **Ultra** | 7+ | 8+ | Full + iteration | Existential decisions, identity shifts |

---

## When to Use Each Mode

### 👀 Show Me
**"I want to understand what SPAR does."**

One round, four personas, no abstraction. The fastest way to see the NEWS compass in action. Good for demos and first-timers. Not for real decisions.

### ⚡ Quick
**"I need a sanity check on this decision by end of day."**

Three rounds with the core compass. Only Type 1 decomposition (break the problem into parts). No relational mapping, no reframing. Fast, directional, not exhaustive.

### ⚖️ Basic
**"This matters, but it's not bet-the-company."**

The default for most decisions. Three rounds with PROBE watching for bias. Types 1-2 abstraction gives personas a shared map and relationship awareness. This is where most practitioners operate.

### 💣 Stress Test
**"We're making a significant commitment. I want to break this before we build it."**

Same roster as Basic, but adds Type 3 abstraction (reframe the problem). The reframe often reveals that the group is debating the wrong question. Use for budget decisions, hiring, partnerships.

### 🧠 Deep
**"This is a strategic pivot. We need every angle."**

Five rounds with the full PERSONALITY roster (8+ agents). Extended debate lets positions evolve across rounds. Steelman is common here: each persona must strengthen the opposing view before attacking it. Use for M&A, market entry, org restructuring.

### 🤯 Ultra
**"This decision defines who we are."**

Seven or more rounds with recursive abstraction. The ABSTRACT phase itself may iterate if the first decomposition proves insufficient. Reserved for existential decisions: identity pivots, values conflicts, irreversible commitments.

## Choosing Your Mode

```
How important is this decision?
│
├── "Just exploring" ────────────→ 👀 Show Me
├── "Need a quick gut check" ────→ ⚡ Quick
├── "Standard decision" ─────────→ ⚖️ Basic
├── "Significant investment" ────→ 💣 Stress Test
├── "Strategic pivot" ───────────→ 🧠 Deep
└── "Defines who we are" ────────→ 🤯 Ultra
```

---

## Escalation

You can always escalate mid-debate. If a Quick debate surfaces unexpected complexity, upgrade to Basic or Stress Test. The key signals:

- **New dimensions emerged** that weren't in the original SCOPE
- **Personas are splitting** on a fundamental value, not just tactics
- **The abstraction map** reveals relations you didn't anticipate
- **PROBE flags** a systematic bias in the group
