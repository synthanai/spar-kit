# DOORS: Decision Exit Framework

*When is a SPAR done? When can you walk away?*

> **Version**: 1.0 (Protocol v8.0)

---

## Overview

DOORS defines the exit conditions for a SPAR debate. Not every tension needs resolution. Not every SPAR needs a verdict. DOORS tells you when you've extracted enough value to decide, and when it's safe to stop.

---

## The DOORS Framework

| Letter | Dimension | Question |
|--------|-----------|----------|
| **D** | Decision Owner | Who makes the final call? Is that person in the room? |
| **O** | Open-Ended Resolution | What stays unresolved, and is that acceptable? |
| **O** | Outstanding Risks | What risks remain after the debate? Are they mitigated? |
| **R** | Ready Signals | What conditions indicate we have enough information to decide? |
| **S** | Safe Walk-Away | What does "good enough" look like? Can we stop here? |

---

## Using DOORS in Practice

### During INTERROGATE

Before moving to TRANSMIT, run the DOORS check:

```
DOORS Check:
  D - Decision Owner: [Name]. Present? [Yes/No]
  O - Open-Ended: [List unresolved tensions]. Acceptable? [Yes/No]
  O - Outstanding Risks: [List remaining risks]. Mitigated? [Yes/No/Partially]
  R - Ready Signals: [List what we now know that we didn't before]
  S - Safe Walk-Away: [Can we decide with current information?] [Yes/No]
```

### Decision Matrix

| DOORS Result | Action |
|-------------|--------|
| All Yes | → TRANSMIT (proceed to verdict) |
| D = No (owner absent) | → Pause. Present findings when owner available. |
| O = No (key tension unresolved) | → Additional RUMBLE round, or Arena Mitosis |
| R = No (not enough information) | → Research gap. Pause SPAR, gather data, resume. |
| S = No (stakes too high for current confidence) | → Escalate depth mode or add domain experts |

---

## Ready Signals by Depth Mode

| Depth Mode | Minimum Ready Signals |
|------------|----------------------|
| 👀 Show Me | 1 new insight surfaced |
| ⚡ Quick | Clear directional recommendation |
| ⚖️ Basic | 2+ convergences, tensions mapped |
| 💣 Stress Test | Pre-mortem completed, risks named |
| 🧠 Deep | All ASPIRES checks pass |
| 🤯 Ultra | Full DOORS pass + PROBE satisfied |

---

## When NOT to Close

Even if DOORS check passes, do NOT close if:
- PROBE has an unresolved objection
- Centre (Behavioral Analyst) flags an adoption risk that hasn't been addressed
- The decision owner says "I need to think about it" (respect the pause)
- New information arrived during the debate that hasn't been incorporated
